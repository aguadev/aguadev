===================
dijit.PopupMenuItem
===================

.. contents ::
    :depth: 2

This is a child of a  :ref:`dijit.DropDownMenu <dijit/DropDownMenu>` or
:ref:`dijit.Menu <dijit/Menu>` which, when clicked, opens a sub-menu.

See the :ref:`dijit.DropDownMenu <dijit/DropDownMenu>` page for examples.
